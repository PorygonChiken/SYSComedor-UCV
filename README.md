# SYSComedor-UCV
Proyecto para la asignatura de Ingeniería de Software enfocado en la gestión del Comedor de la UCV.
El objetivo fue crear un sistema funcional aplicando el patrón MVC (Modelo-Vista-Controlador).

# Tecnologías
Lenguaje: Java (JDK 17).
Interfaz: Swing (Nativo).
Arquitectura: MVC manual (sin frameworks).
Simulación de Verificación Facial: API estándar de Java NIO para la comparación byte a byte de imágenes.
